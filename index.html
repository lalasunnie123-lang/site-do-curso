Tim Berners-Lee

Sir Tim Berners-Lee

O Inventor da World Wide Web

Suas Invenções Fundamentais

HTML

Criou a Linguagem de Marcação de Hipertexto, a espinha dorsal para a criação de páginas web.

HTTP

Desenvolveu o Protocolo de Transferência de Hipertexto, que permite a comunicação entre clientes e servidores na web.

URL

Padronizou o Localizador Uniforme de Recursos, o sistema de endereçamento que usamos para encontrar recursos na internet.
